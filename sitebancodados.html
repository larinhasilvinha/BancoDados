Disciplina: Banco de Dados

Nesta disciplina, você aprenderá os conceitos fundamentais de banco de dados, incluindo modelagem de dados, manipulação de dados usando SQL (Structured Query Language) e design de bancos de dados relacionais.

Banco de Dados é uma área essencial para qualquer pessoa que trabalhe com desenvolvimento de software, análise de dados, sistemas de informação, entre outras áreas relacionadas.

Informações do Curso

Pré-requisito: Conhecimentos básicos em programação.

Carga horária: Total (80 h/a - 60 h/r) AT (40 h/a - 30 h/r) AP (40 h/a - 30 h/r)

Ementa: Capacitar o discente no desenvolvimento, instalação, gerenciamento e manipulação de bancos de dados estruturados e não-estruturados em SGBDs com comunicação em sistemas de informação.
